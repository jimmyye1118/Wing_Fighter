
## <b>作品名稱: </b>Wing_Fighter
## <b>程式語言: </b>C++
## <b>程式介紹: </b> 
### 一個簡單的飛機遊戲，發射子彈，攻擊對方，獲取分數，含有背景音樂，發射子彈的聲音。
## <b>程式玩法: </b>
### 用左右鍵控制飛機移動，按下空白鍵發射子彈，攻擊對方，獲取分數。
